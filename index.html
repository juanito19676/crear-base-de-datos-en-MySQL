Bases de datos

Las bases de datos son algo que debemos de tener muy en cuenta cuando queremos almacenar nuestros datos con seguridad, que esten bien distribuidos y por supuesto modificarlas cuando tengamos que hacerlo.

En especial las bases de datos son algo que vemos todos los dias pero que no sabemos como usarlas por eso es que se usan distintos metodos pero en especial se usan los gestores de bases de datos(SGBD) que nos ayudan a administrar todo siempre y cuando sepamos lo que se hace en cada caso. En este caso estaremos usando el gestor MySQL.

MySQL

Es un gestor que nos ayuda administrar la informacion que nos proveen de la entidad relacion que se puede relacionar con diferentes componentes que tengan relacion entre ellos.

Debemos de usarlo para fines distintos a lo establecido.

    Crear bases de datos

    Como vemos la instalacion de MySQL es importantes ya que sin ellas no podemos hacer nada en cuanto al uso de una base de datos, pero eso no es todo lo que debemos de estar conscientes es que el uso de las bases de datos necesita de ciertos comandos que nosotros podemos usar, y en primer lugar se necesitan los comandos basicos que son para crear una base de datos. Y para ello se emplean los comandos siguientes:

    Comandos

    Create database

    Como su nombre dice este es el comienzo de todo ya que con esta codigo podemos dara a pie a todo lo que nosotros queremos crear y de este paso podemos crear todas las tablas.

    Use database

    Esta sentencia es la continuacion de la sentencia anterior debido a que una vez creada la tabla o que queramos usarla cuando ya esta completada con esto le damos el permiso de poder modificarla a nuestro gusto.

    Drop database

    En caso de que nosotros no queramos usar la base de datos o por algun motivo queremos que ya no exista simplemente le damos esta sentencia que ayudara a eliminarla por completo la cual se debe de tener mucho cuidado con ella ya que si esto sucede toda la base de datos sera boorada y no habla forma de regresarla a menos de que volvamos a poner las funciones anteriores.

    Crear tablas

    En todas las bases no debe de faltar las tablas que como vemos al igual que las sentencias anteriores son parte de MySQL pero que se especializan mas en poner datos.

    Tambien debemos de estar de acuerdo que existen varias cosas extra que se pueden colocar en esta parte como lo serian los componentes que definen por completo como es que esta reprentado un elemento.

    Comandos

    Descripci칩n

    Create table

    Como su nombre dice el uso de las tablas se presenta despues de la creacion de las bases de datos con el cual podemos dar uso de las tablas y por tal motivo podemos tener elementos dentro de ella.

    drop table

    Igualmente si se desea eliminar cierta tabla que no queramos usamos esta sentencia, como en la drop database se debe tener cuidado al emplearla.

    Rename table

    Este a diferencia de los dos anteriores solo permitira cambiar los nombres por lo que si no estamos conformes con la tabla pero no queremos borrarlo simplemente basta con que usemos esta sentencia.

    Int

    Este codigo permitira que los datos que se vayan poniendo dentro de las tablas sean enteros por lo cual esta forma sera mas para el uso de numeros.

    Int not null_autoincrement

    En esta parte podemos dar uso a lo mismo del int pero que aqui este tendra un numero automaticamente conforme vayan apareciendo datos.

    varchar, char, text

    Aqui se muestra el tipo de texto que puede tener los datos por lo que aqui podemos poner datos como gmail o nombres con el fin de llevar cabo una buena administracion.

    Datetime

    Si nuestra base de datos se requiere saber el tiempo que se vayan poniendo los datos esta sentencia ayuda mucho por lo que ya viendo datos mas especificos pueden cambiar.

    La sentencias para esto son las siguientes:

    Insert. insert into

    Esta sentencia te permite insertar datos, los cuales son de las tablas que se han creado por lo que depende cuantos datos se hayan puesto en este caso despues de poner el codigo ponemos los datos de la tabla y seguido de la sentencia values para agregar los datos que queramos, hay que estar deacuerdos que si falta algun dato sera dificil que se haya insertado un dato.

    delete

    Una vez hayamos insertado los datos por si algun motivo nosotros pusimos un dato por error la forma mas sencilla de eliminarlo es usando esta sentencia la cual esta se puede eliminar por medio de alguna caracteristica que tenga ese dato como la edad, por lo que se debe tener cuidado al querer eliminar un dato si no se quiere eliminar algo correcto.

    Update

    En este caso si el dato es erroneo pero igualmente no es para que se borre por lo que esta sentencia se encarga de modificar lo que se desea ya sea si el nombre esta mal o incluso el salario que puede tener un trabajador que reciba un aumento, entre otras cosas.
